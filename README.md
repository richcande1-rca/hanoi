# HANOI

A neon-night Tower of Hanoi game.

## Core rule

Move one disc at a time. Never place a larger disc on a smaller disc.

## Scoring

Moves are the primary score. The mathematical minimum is `2^n - 1`.

Time begins on the first valid move and is recorded only as a bonus. Best perfect-solution times are stored locally in the browser.

## Run

Open `index.html` in a browser. No build step or dependencies are required.
